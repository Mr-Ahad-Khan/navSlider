# 🚀 Modern Responsive Landing Page

A sleek and responsive landing page built with **React**, **Tailwind CSS**, and **Swiper**. This project features a fullscreen hero slider, feature showcase, smooth animations, and call-to-action sections, making it perfect for startups, portfolios, agencies, or business websites.

---

## ✨ Features

- 🎯 Fully Responsive Design
- 🎨 Modern & Clean UI
- 🖼️ Fullscreen Hero Slider
- ⚡ Built with React
- 💨 Styled using Tailwind CSS
- 🎠 Swiper Carousel Integration
- 📱 Mobile-Friendly Layout
- 🔥 Reusable Components
- 🚀 Fast Performance
- 🎭 Smooth Animations
- 📦 Easy to Customize

---

## 🛠️ Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🎠 Swiper.js
- 📜 JavaScript (ES6+)

## 🚀 Getting Started

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Mr-Ahad-Khan/Landing-Page.git
```

### 2️⃣ Navigate to the Project

```bash
cd Landing-Page
```

### 3️⃣ Install Dependencies

```bash
npm install
```

### 4️⃣ Start the Development Server

```bash
npm run dev
```

Open your browser and visit:

```
http://localhost:5173
```
